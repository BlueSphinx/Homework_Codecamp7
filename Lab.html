2. Differentiation Tag
Tag Similarity Difference
<hr>&<br> 1.Both of them give line break.
2.Both of them do not require close tag. 		<hr> gives a horizontal link break but <br> gives an empty line break
<ul>&<ol> 1.Both of them define list.
2.Both of them require close tag. 		<ol> gives ordered list that can be numerical or alphanetical but <ul> gives unordered list.
<select>&<checkbox>&<radio button> All of them provide choices for the user to choose something. 1. <select> creates drop-down list.
2. <checkbox> is shown as a square box that is ticked when activaterd.
3. <radio button> is presented in radio group. Only one radio button in a group can be selected at the same time.
<div>&<span> Both of them define section or division in HTML. <div> is used as a container for HTML elements while <span>is an inline container used to mark up a part of a text, or a part of a document.
<script>&<style> They both embedded external documents. <script> is used to embed a client-side script (JavaScript) while <style> is used to define style information (CSS) for a document.


3. Input Type
Input Type Description
button It defines a button.
checkbox It defines a checkbox that let a user select ZERO or MORE options of a limited number of choices.
color It is used for input fields that should contain a color.
date It is used for input fields that should contain a date.
datetime-local It specifies a date and time input field, with no time zone.
email It is used for input fields that should contain an e-mail address.
file It defines a file-select field and a "Browse" button for file uploads
hidden It defines a hidden input field that let web developers include data that cannot be seen or modified by users when a form is submitted.
image It defines an image as a submit button. The path to the image is specified in the src attribute.
month It allows the user to select a month and year
number It defines a numeric input field. The user can also set restrictions on what numbers are accepted.
password It defines a password field.
radio It defines a radio button which let a user select ONLY ONE of a limited number of choices.
range It defines a control for entering a number whose exact value is not important (like a slider control). Default range is 0 to 100.
reset It defines a reset button that will reset all form values to their default values.
search It is used for search fields (a search field behaves like a regular text field).
submit It defines a button for submitting form data to a form-handler.
tel It is used for input fields that should contain a telephone number.
text It defines a one-line text input field.
time It allows the user to select a time (no time zone).
url It is used for input fields that should contain a URL address.
week It allows the user to select a week and year.
