Website Structure

The first website is "matichon"

  1. Setup a hierarchy structure for main and minor news topics.
  2. Use Grid Layout to seperate the areas according to the main news topics.
  3. The minor news topics will be put inside Grid Container as a Grid Items.
  4. Decorate the content with CSS.

The second website is "webboard kratuu of dek-d.com"

  1. Setup a grid structure for different topics.
  2. Use Grid Layout to seperate the areas according to the main topics.
  3. The minor news topics will be put inside Grid Container as a Grid Items.
  4. Decorate the content with CSS.

The third website is "gamesrig.com"

  1. Setup a grid structure for different headings.
  2. Use Flex Layout to seperate the areas according to the main headings.
  3. The minor items will be put inside the sub-containers then they will be put in the main flex container.
  4. Decorate the content with CSS.
